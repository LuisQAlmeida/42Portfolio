> My journey through the 42 curriculum at **42 Porto**.  
This repository indexes all the projects I completed as part of the **Common Core**.

## Index
### Common Core Projects
<details>
<summary><strong> Milestone 0 - Building a library in C</strong></summary>
  
- [Libft](https://github.com/LuisQAlmeida/42Libft) - Custom C library reimplementing standard functions and useful utilities for strings, memory, lists, and more.
</details>

<details>
<summary><strong> Milestone 1 - Foundations</strong></summary>
  
- [ft_printf](https://github.com/LuisQAlmeida/42ft_printf) - Reimplementation of the printf function supporting formatted output with a subset of the C standard specifiers.
- [get_next_line](https://github.com/LuisQAlmeida/42get_next_line) - Line-reading utility that returns one line at a time from a file descriptor, handling arbitrary buffer sizes.
- [Born2beroot](https://github.com/LuisQAlmeida/42Born2beroot) - System administration project that sets up and secures a virtual machine with strict user, partition, and service configuration.
</details>

<details>
<summary><strong> Milestone 2 - Signals, Algorithms & Graphics</strong></summary>
  
- [minitalk](https://github.com/LuisQAlmeida/42minitalk) - Small client/server program that sends and receives strings using UNIX signals (SIGUSR1/SIGUSR2) as a custom communication protocol.
- [push_swap](https://github.com/LuisQAlmeida/42push_swap) - Stack-based sorting program that orders integers using a limited set of operations while minimizing the number of moves.
- [so_long](https://github.com/LuisQAlmeida/42so_long) - Simple 2D game using MiniLibX where the player navigates a map, collects items, and reaches the exit with valid path checking.
</details>
